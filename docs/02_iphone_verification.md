# iPhone実機確認

## 公開前に必要な判断

ホーム画面へ追加できるPWAとして使うにはHTTPS公開が必要になる。次のどちらかを選ぶ。

1. GitHubの公開リポジトリとCloudflare Pages
2. 非公開リポジトリを使える別のHTTPS配信環境

個人情報や医療記録は含まず、履歴はiPhone内のlocalStorageだけに保存する。公開されるのはアプリのコードと画面のみ。

## 初回確認

1. iPhoneのSafariで公開URLを開く
2. 共有メニューから「ホーム画面に追加」
3. ホーム画面から起動
4. カード短辺を使って画面サイズを校正
5. 裸眼、画面から約40cm、普段の室内照明で使用

## 標準トレーニング

- 1回目と2回目の提示が明確に分かれる
- 中央の薄い縞が常に見えるわけではない
- 正解を重ねると徐々に薄くなる
- 不正解後は少し見やすくなる
- 縞の周辺が四角く切れず、灰色へ滑らかに溶ける
- ノイズマスクが一瞬表示される

## 絵探しゲーム

- 4×4の全パッチが画面幅に均等に並ぶ
- 見本と同じパッチが必ず1つある
- タップ後の反応が遅れない
- 誤タップ時に短い振動と枠表示がある

## 終了後

- 疲労度を入力できる
- 20秒休憩が終わるまで完了ボタンを押せない
- 完了後、今日の件数と履歴が更新される
- 機内モードでも2回目以降は起動できる

## 中止基準

次のいずれかがあれば、その日の使用を中止する。

- 眼痛
- 複視
- 頭痛や悪心
- 急なかすみ
- 片眼だけの見え方の変化
- 使用後も続く強い眼精疲労
